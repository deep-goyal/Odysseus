It includes syntax highlighting, auto-formatting, and much more. A few popular IDEs are VS Code, Eclipse, intelliJ, and XCode. Odysseus requires you to use Visual Studio Code. Command Line Interface and the Bourne Again SHell (BASH)

Now that you have a clean setup, lets also get used to the command line, an interface that you will inevitably use quite often. The Software Carpentry's Unix Shell Document↗ provides a comprehensive introduction to the language. I would suggest reading the chapters 1-4, until you are familiar with pipers and filters. I would also suggest watching this video↗ about BASH once you are familiar with the command line, and how to use the shell in general. One thing to note, BASH and command line are not the same thing. Command line is just a text based interface, and BASH is a shell that provides this interface and many other scripting tools that you can leverage to run your programs efficiently.

What is programming?

Programming is the art of creating instructions, known as code, that computers can understand and execute. It's like speaking a special language to communicate with our electronic friends. We use programming languages like C, Java, or Python to write logical and structured commands that solve problems and perform tasks. Programmers are like puzzle solvers! They break down complex problems into smaller, manageable steps, using their creativity and logical thinking. It's a bit like following a recipe or solving a puzzle, but with a tech twist! After we write our code, it goes through a fascinating transformation. First, it's translated into assembly code, which is a more readable version for computers. Then, it's further translated into machine code—instructions that the computer's hardware can directly understand. Finally, it becomes a binary executable—the ultimate language that our computer loves! Programming is more than just automation and data manipulation; it's a versatile skill that opens up a world of possibilities. With programming, we can automate tasks, manipulate data, build software applications, and much more. It requires practice and attention to detail, allowing us to harness the power of computers to accomplish various goals.

Here is your first task. You have to learn a general purpose programming language. It could be C, C++, Python, or Java.
